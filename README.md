# ✈ High Cloud Airline Performance Dashboard

## 📌 Project Overview
This project analyzes airline operational data to evaluate performance trends in seat utilization, passenger volume, route demand, and carrier efficiency.

The dashboard was built using Microsoft Excel with Power Query, Data Modeling, and DAX.

---

## 📊 Key Performance Indicators
- Total Flights: 2.8M+
- Passenger Volume: 187M+
- Load Factor: 76.8%
- Avg Passengers per Flight: 67
- Capacity Utilization Gap: 56.5M seats

---

## 📈 Dashboard Features
- Yearly Load Factor Trend
- Carrier-wise Load Factor Comparison
- Weekend vs Weekday Performance
- Top 5 Busiest Routes
- Top 10 Carriers by Passenger Volume
- Interactive Slicers for dynamic filtering

---

## 🛠 Tools & Technologies
- Excel
- Power Query
- Power Pivot (Data Model)
- DAX Measures
- Pivot Tables & Charts

## 💡 Business Insights
- Weekend load factor slightly exceeds weekday demand.
- Peak performance observed in 2013.
- 56.5M seats remain underutilized — indicating optimization potential.
- Certain carriers consistently maintain 85%+ seat utilization.
